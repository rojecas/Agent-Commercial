# Glosario del Proyecto

Este documento define los términos clave utilizados en el proyecto para asegurar que tanto humanos como agentes utilicen un lenguaje común.

| Término | Definición | Contexto/Uso |
| :--- | :--- | :--- |
| **Agentic Workflow** | Flujo de desarrollo optimizado para la asistencia de IA (Antigravity). | Metodología |
| **Spec-Driven** | Desarrollo basado en especificaciones previas a la codificación. | Metodología |
| **A2A** | Protocolo Agent-to-Agent para comunicación entre agentes. | Tecnología |
| **MCP** | Model Context Protocol para conexión con herramientas. | Tecnología |
