INASC — Widget de Chat

Esta página demuestra el Asistente Virtual de INASC funcionando en tiempo real a través de WebSocket. El widget aparece en la esquina inferior derecha de cualquier página donde se integre.

Integración en www.inasc.com.co
Para integrar el widget en cualquier página, añade estas dos líneas antes de cerrar el tag </body>:

<link rel="stylesheet" href="/static/widget.css">
<!-- [pegar el bloque HTML del widget] -->
<script src="/static/widget.js"></script>
